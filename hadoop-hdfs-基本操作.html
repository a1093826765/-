（与linux命令原理相似）
  1. hadoop fs -ls 目录地址
  2. hadoop fs -put 上传文件地址 目录地址
    1. hadoop -fs -copyFromLocal 上传文件地址 目录地址
  3. hadoop fs -get 目录文件地址 本地地址
    1. hadoop fs -copyToLocal 目录文件地址 本地地址
  4. hadoop fs -cat 目录文件地址
  5. hadoop fs -mkdir 文件夹名
  6. hadoop fs -moveFromLocal 本地文件地址 目录地址
  7. hadoop fs -moveToLocal 本地地址 目录文件地址
  8. hadoop fs -appendToFile 本地文件地址 目录文件地址
  9. hadoop fs -tail 目录文件地址 
  10. hadoop fs -text 目录文件地址
  11. hadoop fs -chgrp 组名 目录文件地址
  12. hadoop fs -chmod 权限值 目录文件地址
  13. hadoop fs -chwon用户:所属组 目录文件地址
  14. hadoop fs -cp 目录源文件地址 目录目标地址
  15. hadoop fs -mv 目录源文件地址 目录目标地址
  16. hadoop fs -getmerge 目录多个文件地址（用*匹配） 本地目标文件地址
  17. hadoop fs -rm -rf 目录文件地址
  18. hadoop fs -rmdir 目录地址
  19. hadoop fs -df -h 目录地址
  20. hadoop  fs  -du  -s  -h 目录地址（如果要统计根目录则用：hdfs://主机名:9000/*）
  21. hadoop fs -count 目录地址
  22. hadoop fs -setrep 数量 目录文件地址（实际数量的最大值为集群datanode的最大值）
  23. hdfs dfsadmin -report
