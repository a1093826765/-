笔记Notes
  1. 定义一个变量到hive，可用于hql语句中，${key}
  2. 指定一个使用的数据库
  3. 执行命令行获取的一条sql，执行返回结果后自动退出（通常用于脚本中）
  4. 执行文件fileName中的sql语句，执行返回结果后自动退出
  5. 执行文件fileName中的sql语句，执行返回结果后不退出
  6. Hive在运行时，先读取hadoop的配置文件，再读取hive-default.xml，再读取hive-site.xml。如果使用- -iveconf可以定义一组属性，这个属性会覆盖之前读到的参数值
  7. 不答应与结果无关的信息
  8. 输入到控制台
  9. 在hive中使用hadoop命令
  10. 在hive中使用linux命令
  11. 设置本地模式运行
  1. Hive -d key=value（hive - -hivevar key=value）
  2. Hive - -database databaseName
  3. Hive -e ’sql'
  4. Hive -f fileName
  5. Hive -i fileName
  6. Hive -hiveconf property=value
  7. Hive -S
  8. Hive -V
  9. Fds （hadoop命令）
  10. !（linux命令）
  11. 设置本地模式运行：(只在reduceTask个数为1时有效)
    1. Set hive.exec.mode.local.auto=true;//开始本地mr
    2. Set hive.exec.mode.local.auto.inputbytes.max=xxx；//设置当输入数据量小于此值使用本地模式
    3. Set hive.exec.mode.local.auto.input.files.max=xx; //设置当输入文件个数小于此值使用本地模式
