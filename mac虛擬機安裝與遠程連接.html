  1. 終端  yum install net-tools.x86_64 

    1. 終端 cd /etc/sysconfig/network-scripts/
    2. 打開 ifcfg-ensXX 的文件
    3. 修改 ONBOOT=yes
    4. 重啟網卡 service network restart
    5. 關閉防火牆 systemctl stop firewalld.services
    6. 設置開機自動關閉防火牆 systemctl disable firewalld.service
  3.  
    1. 打開 /etc/ssh/sshd_config
    2. 修改 PermitRootLogin yes
    3. 遠程連結 ssh root@地址 
  4.  
    1. Mac传送目录到Linux服务器(文件上传无需-r) scp -r 源文件地址 root@地址:目標文件地址
    2. Linux服务器下载文件到Mac                             scp root@地址:源文件地址 目標地址
