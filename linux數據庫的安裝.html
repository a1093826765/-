笔记Notes
  1. Mysql 數據庫的安裝
  1. Centos: 
    1. 進入http://repo.mysql.com/根據所需要的mysql下載安裝包
    2. wget http://repo.mysql.com/mysql57-community-release-el7-11.noarch.rpm
    3. rpm -ivh mysql57-community-release-el7-11.noarch.rpm
    4. 進入 etc/yum.repos.d/ 目錄執行
    5. yum install mysql-server
    6. 第一次登陸，獲取臨時密碼 grep 'temporary password' /var/log/mysqld.log
    7. 登陸之後修改密碼  use mysql;                                update mysql.user set authentication_string=password(‘密碼') where user='root' ;
    8. 立即生效 flush privileges;
