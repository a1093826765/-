  13. Select * from table where name in(‘xxx’,’xx')
  14. 不同的数据库对 BETWEEN...AND 操作符的处理方式是有差异的。某些数据库会列出介于 "name" 和 “name1" 之间的人，但不包括 "name" 和 “name1" ；某些数据库会列出介于 “name" 和 “name1" 之间并包括 “name" 和 “name1" 的人；而另一些数据库会列出介于 “name" 和 “name1" 之间的人，包括 “name" ，但不包括 “name1" 。所以，请检查你的数据库是如何处理 BETWEEN....AND 操作符的！
    1. Select * from table where name between ‘xxx’ and ‘xx'
    2. 反向 范围之外：select * from table where name not between ‘xxx’ and ‘xx'
  15.  别名

  16.  两表之间关系查询
    1. 如果表中有至少一个匹配，则返回行：（inner join）交集：select * from table inner join table1 on table1.name=table.name
    2. 即使右表中没有匹配，也从左表返回所有的行：（left join）left完全集，右边没有则用null：select * from table left join table1 on table1.name=table.name
    3. 即使左表中没有匹配，也从右表返回所有的行：（right join）right完全集，左边没有则用null：select * from table right join table1 on table1.name=table.name
    4. 只要其中一个表中存在匹配，就返回行：（full join）完全集，没有则用null：select * from table full join table1 on table1.name=table.name
  17. 把两表查询合并
    1. 只合并唯一值：select name from table union select name from table1
    2. 合并做所有：select name from table union all select name from table1

  21.  约束
    1. $1：not null — 非空
    2. $2 / $1：unique — 唯一约束
      1. Mysql：只能放在$2 unique(列名)
      2. SQLserver/Oracle/MSaccess：只能放在$1 unique
      3. 命名约束以及多个列设置唯一：放在$2  constraint 命名 union(列名，列名) 
      4. 增加表唯一约束：alter table tablename add union(列名，列名)
        1. 如需命名：alter table add constraint 命名 union(列名，列名)
      5. 删除表唯一约束
        1. Mysql：alter table drop index 约束名
        2.  SQLserver/Oracle/MSaccess：alter table drop constraint 约束名
    3. $2 / $1：primary key — 主键
      1. Mysql：只能放在$2 primary key (列名)
      2. SQLserver/Oracle/MSaccess：只能放在$1 primary key
      3. 命名约束：放在$2 constraint 命名 primary key (列名)
      4. 增加表主键：alter table add primary key (列名)
        1. 如需命名：alter table add constraint 命名 primary key (列名)
      5. 删除表主键
        1. Mysql：alter table drop primary key
        2. SQLserver/Oracle/MSaccess：alter table drop constraint 主键约束名
    4. $2 / $1：foreign key — 外键
      1. Mysql： 只能放在$2 foreign key references table1(列名) 
      2. SQLserver/Oracle/MSaccess：只能放在$1 foreign key references table1(列名)
      3. 命名约束：放在$2 constraint 命名 foreign key (列名) references table1(列名)
      4. 增加表外建：alter table tablename add constraint 命名 foreign key (列名) references table1(列名)
      5. 删除外键：
        1. Mysql：alter table tablename drop foreign key 外键约束名
        2.  SQLserver/Oracle/MSaccess：alter table tablename drop constraint 外键约束名
    5. $2 / $1：check — 自定限制约束
      1. Mysql： 只能放在$2 check (id>0)
      2. SQLserver/Oracle/MSaccess：只能放在$1 check (id>0)
      3. 命名约束：放在$2 constraint 命名 check (id>0 and type=‘xxx')
      4. 增加限制约束：alter table tablename add constraint 命名 check (id>0 and type=‘xxx')
      5. 删除外键：
        1. Mysql：alter table tablename drop 限制约束名
        2.  SQLserver/Oracle/MSaccess：alter table tablename drop constraint 限制约束名
    6. $1：default — 默认值约束
      1. 只能放在$1 default ‘xxx'
      2. 增加默认值约束：
        1. Mysql ：alter table tablename alter 列名 set default ‘xxx'
        2. SQLserver/Oracle/MSaccess：alter table tablename alter column 列名 set   default ‘xxx'
      1. 删除外键：
        1. Mysql：alter table tablename alter 列名 drop default
        2.  SQLserver/Oracle/MSaccess：alter table tablename alter column 列名 drop default
