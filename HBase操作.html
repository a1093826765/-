  1. 启动hbase
    1. 在每台机器上启动 regionserver ：sh hbase-daemon.sh start regionserver
    2. 在任意一台机器启动 master：sh hbase-daemon.sh start master
  2. 停止hbase
    1. 在每台机器上停止 regionserver ：sh hbase-daemon.sh stop regionserver
    2. 在master机器停止 master：sh hbase-daemon.sh stop master
  3. 群启动hbase
    1. 在任意一台机器启动 master：sh hbase-daemon.sh start master
    2. sh hbase-daemons.sh start regionserver
  4. sh stop-hbase.sh
