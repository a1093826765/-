笔记Notes

  2. 插入mysql配置文件，免密登陸
  3. 重啟mysql服務
  4. 修改密碼腳本
  1. service mysqld stop
  2. sed -i "5i skip-grant-tables" /etc/my.cnf
  3. service mysqld start
  4. MYSQL=$(which mysql)
    1. $MYSQL -u root -e “use mysql; update user set authentication_string=passworD('a2288311') where user='root'; flush privileges;” (5.7以上版本)
    2. $MYSQL -u root -e “UPDATE user SET password=password(‘a2288311') WHERE user='root’; flush privileges;" (5.6以下版本)
