  1. Show databases;
  2. Show tables in 库名;
  3. Desc database 库名;
    1. Desc database extended 库名; — 查看库的详细信息
  4. Create database if not exists（这里是判断库是否存在）库名 comment 库的注释 location 库在hdfs路径 with dbproperties (property_name=property_value) ;（这里是库的属性）
  5. Use 库名;
  6. 修改库
    1. 修改属性：alter database 库名 set dbproperties (property_name=property_value);
    2. 修改使用人：alter database 库名 set owner 使用人;
    3. 修改hdfs地址：alter database 库名 set location hdfs地址
  7. Drop database 库名;（只能删除空库）
    1. Drop database 库名 cascade;（删除库，可删除非空库）
