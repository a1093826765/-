笔记Notes
  1. jdk的安裝
  2. 環境變量的設置
  3. 刷新配置，使配置生效
  1.  
    1. Centos:  yum install -y java-11-openjdk*
    2. Ubuntu:  apt-get install java-11-openjdk.x86_64
  2.  配置環境變量 /etc/profile
    1. export JAVA_HOME=/usr/lib/jvm
    2. export JRE_HOME=${JAVA_HOME}/jre
    3. export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib:$CLASSPATH
    4. export JAVA_PATH=${JAVA_HOME}/bin:${JRE_HOME}/bin
    5. export PATH=$PATH:${JAVA_PATH}
  3. 配置生效 source /etc/profile
